# 🍔 Food Ordering App (React + Vite + TypeScript)

A modern, scalable food ordering frontend application built using React, Vite, TypeScript, and Tailwind CSS.

---

## 🚀 Features

- OTP-based login with JWT authentication
- Backend-synced cart
- Real-time menu from API
- Coupon system
- Order & payment processing
- Clean architecture
- Type-safe API calls
- Tailwind CSS UI

---
